# Family Graduation Diagnostics

`plan-family-graduation` is a saved-file-only planning report. It does not fetch sources, trust registry proposals, change settlement-evidence burden rows, create exact-payoff rows, create evaluator rows, or emit paper candidates.

Registry proposals in this report are not evidence. A human reviewer must validate the source, scope, typed-key requirements, evidence quote or excerpt, limitations, reviewer identity, review timestamp, and expiry before a registry entry can support source or exact-payoff review.

Registry entries cannot bypass typed-key matching, same-payoff review, quote/depth/freshness checks, fee checks, relationship gates, or evaluator gates.
